git init > criar o repositório local

git remote add origin "link git hub" > Conecta o repositório local com o github

git add. > adiciona tudo da pasta ao repositório

git commit -m "mensagem do commit" > Faz o commit do versionamento

git push origin main > Envia o versionamento para o GitHub

************Branch Nova*********************

git branch > mostra todas as branchs

git checkout > sai da branch no git

git checkout -b > cria um nova branch
